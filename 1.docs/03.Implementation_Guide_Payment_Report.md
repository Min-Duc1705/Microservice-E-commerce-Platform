# Hướng dẫn hoàn thiện PaymentService & ReportService

---

## ❓ Saga Pattern là gì? Có nên dùng không?

### Khái niệm

**Saga Pattern** giải quyết vấn đề: _"Làm sao đảm bảo tính toàn vẹn dữ liệu khi một luồng nghiệp vụ chạy qua nhiều Microservice?"_
h
Ví dụ luồng đặt hàng của dự án này:

```
Tạo Order → Tạo Payment → Trừ kho ProductService → Cộng TotalSpent CustomerService
```

Nếu bước 3 thất bại → Order đã tạo, Payment đã tạo → dữ liệu không nhất quán.

Saga giải quyết bằng **Compensating Transactions** (giao dịch bù trừ): mỗi bước đều có bước ngược để rollback.

### 2 loại Saga

|                    | Choreography (đang dùng)                   | Orchestration                         |
| ------------------ | ------------------------------------------ | ------------------------------------- |
| **Cách hoạt động** | Mỗi service tự lắng nghe event và phản ứng | Có 1 Orchestrator trung tâm điều phối |
| **Ưu điểm**        | Đơn giản, loose coupling                   | Dễ debug, logic tập trung             |
| **Nhược điểm**     | Logic phân tán, khó theo dõi toàn luồng    | Thêm code phức tạp (StateMachine)     |
| **Phù hợp**        | Dự án nhỏ-vừa                              | Dự án lớn, nghiệp vụ phức tạp         |

### Dự án này có nên dùng không?

**Trả lời: KHÔNG cần Orchestration Saga. Choreography hiện tại là đủ.**

**Lý do:**

1. Dự án đang dùng **Choreography-based Saga** rồi (Event → Consumer → Event → Consumer...)
2. MassTransit Orchestration Saga yêu cầu `StateMachine` phức tạp, overkill cho intern project
3. Hệ thống bán hàng quy mô nhỏ không cần đảm bảo tính ACID phân tán nghiêm ngặt

**Luồng Choreography hiện tại của dự án:**

```
OrderService → OrderCreatedEvent
    ├── PaymentService (consume) → tạo Payment (Pending)
    │       → Admin confirm → PaymentSucceededEvent
    │                         PaymentService → OrderService cập nhật status
    └── ProductService (consume) → Trừ kho khi OrderStatusChangedEvent = Processing
```

namespace CommonService.Events
{
    public class OrderCreatedEvent : IEvent
    {
        public Guid OrderId { get; set; }
        public Guid? CustomerId { get; set; }          // ← THÊM MỚI
        public string CustomerName { get; set; } = string.Empty;
        public string CustomerPhone { get; set; } = string.Empty;
        public decimal SubTotal { get; set; }          // ← THÊM MỚI (tiền hàng)
        public decimal ShippingFee { get; set; }       // ← THÊM MỚI
        public decimal TotalAmount { get; set; }       // SubTotal + ShippingFee
        public string PaymentMethod { get; set; } = string.Empty; // ← THÊM MỚI
        public List<OrderItemInfo> Items { get; set; } = new();
        public DateTime ProcessedAt { get; set; }
    }

    public class OrderItemInfo
    {
        public Guid ProductId { get; set; }
        public string ProductName { get; set; } = string.Empty;
        public int Quantity { get; set; }
        public decimal UnitPrice { get; set; }
        public decimal UnitCost { get; set; }          // ← THÊM MỚI (giá vốn)
    }
}

```csharp
using MassTransit;
using MassTransit.EntityFrameworkCoreIntegration;
using Microsoft.EntityFrameworkCore;
using PaymentService.Models;

namespace PaymentService.Data;

public class PaymentDbContext : DbContext
{
    public PaymentDbContext(DbContextOptions<PaymentDbContext> options) : base(options) { }

    public DbSet<Payment> Payments { get; set; }

    // ── MassTransit Outbox Tables ─────────────────────────────
    public DbSet<OutboxMessage> OutboxMessages { get; set; }
    public DbSet<OutboxState>   OutboxStates   { get; set; }
    public DbSet<InboxState>    InboxStates    { get; set; }

    protected override void OnModelCreating(ModelBuilder modelBuilder)
    {
        base.OnModelCreating(modelBuilder);

        modelBuilder.Entity<Payment>(entity =>
        {
            entity.HasKey(e => e.Id);
            entity.Property(e => e.Amount).HasColumnType("decimal(18,2)");

            // Mỗi Order chỉ có đúng 1 Payment
            entity.HasIndex(e => e.OrderId)
                  .IsUnique()
                  .HasDatabaseName("IX_Payments_OrderId");

            entity.HasIndex(e => e.Status)
                  .HasDatabaseName("IX_Payments_Status");

            entity.HasIndex(e => e.Method)
                  .HasDatabaseName("IX_Payments_Method");

            entity.HasIndex(e => e.CreatedAt)
                  .HasDatabaseName("IX_Payments_CreatedAt");

            // Composite: Filter thanh toán theo Status + Method (báo cáo)
            entity.HasIndex(e => new { e.Status, e.Method })
                  .HasDatabaseName("IX_Payments_Status_Method");
        });

        // ── Map MassTransit Outbox Tables ──────────────────────
        modelBuilder.AddInboxStateEntity();
        modelBuilder.AddOutboxMessageEntity();
        modelBuilder.AddOutboxStateEntity();
    }
}
```

### 📄 File 9: `Specifications/PaymentFilterSpec.cs`

```csharp
using CommonService.Specifications;
using PaymentService.Models;
using PaymentService.Models.Request;
using PaymentService.Utils.Enum;

namespace PaymentService.Specifications;

public class PaymentFilterSpec : BaseSpecification<Payment>
{
    public PaymentFilterSpec(PaymentFilterRequest f)
        : base(p =>
            (!f.Status.HasValue || p.Status == f.Status.Value) &&
            (!f.Method.HasValue || p.Method == f.Method.Value) &&
            (!f.FromDate.HasValue || p.CreatedAt >= f.FromDate.Value) &&
            (!f.ToDate.HasValue   || p.CreatedAt <= f.ToDate.Value))
    {
        if (f.SortBy == "Amount")
        {
            if (f.IsDescending) AddOrderByDescending(p => p.Amount);
            else                AddOrderBy(p => p.Amount);
        }
        else
        {
            if (f.IsDescending) AddOrderByDescending(p => p.CreatedAt);
            else                AddOrderBy(p => p.CreatedAt);
        }

        ApplyPaging((f.PageNumber - 1) * f.PageSize, f.PageSize);
    }
}

/// <summary>Spec chỉ đếm (không paging) — dùng song song với PaymentFilterSpec</summary>
public class PaymentFilterCountSpec : BaseSpecification<Payment>
{
    public PaymentFilterCountSpec(PaymentFilterRequest f)
        : base(p =>
            (!f.Status.HasValue || p.Status == f.Status.Value) &&
            (!f.Method.HasValue || p.Method == f.Method.Value) &&
            (!f.FromDate.HasValue || p.CreatedAt >= f.FromDate.Value) &&
            (!f.ToDate.HasValue   || p.CreatedAt <= f.ToDate.Value)) { }
}
```

### 📄 File 13: `Services/PaymentServiceImpl.cs`

```csharp
using CommonService.Common;
using CommonService.Events;
using CommonService.Exceptions;
using MassTransit;
using PaymentService.Models;
using PaymentService.Models.Request;
using PaymentService.Models.Response;
using PaymentService.Repository.Interface;
using PaymentService.Services.Interface;
using PaymentService.Specifications;
using PaymentService.Utils.Enum;

namespace PaymentService.Services;

public class PaymentServiceImpl : IPaymentService
{
    private readonly IPaymentRepository _paymentRepo;
    private readonly IPublishEndpoint   _publishEndpoint;

    public PaymentServiceImpl(IPaymentRepository paymentRepo, IPublishEndpoint publishEndpoint)
    {
        _paymentRepo     = paymentRepo;
        _publishEndpoint = publishEndpoint;
    }

    public async Task<ResultPaginationDto<PaymentResponse>> GetPagedPaymentsAsync(PaymentFilterRequest filter)
    {
        // Chuẩn hóa datetime sang UTC (PostgreSQL yêu cầu)
        if (filter.FromDate.HasValue)
            filter.FromDate = DateTime.SpecifyKind(filter.FromDate.Value, DateTimeKind.Utc);
        if (filter.ToDate.HasValue)
            filter.ToDate = DateTime.SpecifyKind(filter.ToDate.Value, DateTimeKind.Utc);

        var spec      = new PaymentFilterSpec(filter);
        var countSpec = new PaymentFilterCountSpec(filter);

        var payments   = await _paymentRepo.ListAsync(spec);
        var totalCount = await _paymentRepo.CountAsync(countSpec);

        return new ResultPaginationDto<PaymentResponse>(
            payments.Select(MapToResponse).ToList(),
            filter.PageNumber, filter.PageSize, totalCount);
    }

    public async Task<PaymentResponse> GetByOrderIdAsync(Guid orderId)
    {
        var spec    = new PaymentByOrderIdSpec(orderId);
        var payment = await _paymentRepo.GetEntityWithSpec(spec)
            ?? throw new NotFoundException($"Không tìm thấy thông tin thanh toán cho đơn hàng: {orderId}");

        return MapToResponse(payment);
    }

    public async Task<PaymentResponse> ConfirmPaymentAsync(Guid orderId, ConfirmPaymentRequest request)
    {
        var spec    = new PaymentByOrderIdSpec(orderId);
        var payment = await _paymentRepo.GetEntityWithSpec(spec)
            ?? throw new NotFoundException($"Không tìm thấy thông tin thanh toán cho đơn hàng: {orderId}");

        if (payment.Status == PaymentStatus.Succeeded)
            throw new BadRequestException("Đơn hàng này đã được xác nhận thanh toán trước đó.");

        if (payment.Status == PaymentStatus.Failed)
            throw new BadRequestException("Không thể xác nhận thanh toán đã thất bại.");

        payment.Status          = PaymentStatus.Succeeded;
        payment.TransactionCode = request.TransactionCode;
        payment.Note            = request.Note;
        payment.PaidAt          = DateTime.UtcNow;
        payment.UpdatedAt       = DateTime.UtcNow;

        _paymentRepo.Update(payment);
        await _paymentRepo.SaveChangesAsync();

        // Thông báo để OrderService cập nhật trạng thái đơn hàng
        await _publishEndpoint.Publish(new PaymentSucceededEvent
        {
            OrderId       = payment.OrderId,
            TransactionId = payment.TransactionCode ?? payment.Id.ToString(),
            ProcessedAt   = DateTime.UtcNow
        });

        return MapToResponse(payment);
    }

    public async Task MarkPaymentFailedAsync(Guid orderId, string reason)
    {
        var spec    = new PaymentByOrderIdSpec(orderId);
        var payment = await _paymentRepo.GetEntityWithSpec(spec)
            ?? throw new NotFoundException($"Không tìm thấy thông tin thanh toán cho đơn hàng: {orderId}");

        if (payment.Status != PaymentStatus.Pending)
            throw new BadRequestException("Chỉ có thể đánh dấu thất bại khi đang ở trạng thái Pending.");

        payment.Status    = PaymentStatus.Failed;
        payment.Note      = reason;
        payment.UpdatedAt = DateTime.UtcNow;

        _paymentRepo.Update(payment);
        await _paymentRepo.SaveChangesAsync();

        await _publishEndpoint.Publish(new PaymentFailedEvent
        {
            OrderId     = payment.OrderId,
            Reason      = reason,
            ProcessedAt = DateTime.UtcNow
        });
    }

    private static PaymentResponse MapToResponse(Payment p) => new()
    {
        Id              = p.Id,
        OrderId         = p.OrderId,
        Amount          = p.Amount,
        Method          = p.Method.ToString(),
        Status          = p.Status.ToString(),
        TransactionCode = p.TransactionCode,
        PaidAt          = p.PaidAt,
        Note            = p.Note,
        CreatedAt       = p.CreatedAt
    };
}
```

### 📄 File 14: `Consumers/OrderCreatedConsumer.cs`

```csharp
using CommonService.Events;
using MassTransit;
using PaymentService.Data;
using PaymentService.Models;
using PaymentService.Utils.Enum;

namespace PaymentService.Consumers;

/// <summary>
/// Lắng nghe OrderCreatedEvent → tự động tạo bản ghi Payment với Status = Pending.
/// Admin sẽ xác nhận sau (qua PUT /payments/{orderId}/confirm).
/// </summary>
public class OrderCreatedConsumer : IConsumer<OrderCreatedEvent>
{
    private readonly PaymentDbContext _dbContext;

    public OrderCreatedConsumer(PaymentDbContext dbContext)
    {
        _dbContext = dbContext;
    }

    public async Task Consume(ConsumeContext<OrderCreatedEvent> context)
    {
        var ev = context.Message;

        // Kiểm tra idempotency: tránh tạo trùng nếu event được gửi lại
        var exists = await _dbContext.Payments
            .AnyAsync(p => p.OrderId == ev.OrderId);

        if (exists) return;

        // Parse PaymentMethod từ string trong event
        var method = ev.PaymentMethod switch
        {
            "BankTransfer" => PaymentMethod.BankTransfer,
            _              => PaymentMethod.COD
        };

        var payment = new Payment
        {
            Id        = Guid.NewGuid(),
            OrderId   = ev.OrderId,
            Amount    = ev.TotalAmount,
            Method    = method,
            Status    = PaymentStatus.Pending,
            CreatedAt = DateTime.UtcNow
        };

        await _dbContext.Payments.AddAsync(payment);
        await _dbContext.SaveChangesAsync();
    }
}
```

> **Lưu ý**: Consumer này inject trực tiếp `PaymentDbContext` thay vì dùng Repository vì đây là thao tác đơn giản, không cần Specification. Nếu muốn thuần túy hơn, có thể inject `IPaymentRepository`.

using CommonService.Annotations;
using CommonService.Common;
using Microsoft.AspNetCore.Mvc;
using PaymentService.Models.Request;

using Microsoft.EntityFrameworkCore;
using ReportService.Models;

namespace ReportService.Data;

public class ReportDbContext : DbContext
{
    public ReportDbContext(DbContextOptions<ReportDbContext> options) : base(options) { }

    public DbSet<ReportOrder>     ReportOrders     { get; set; }
    public DbSet<ReportOrderItem> ReportOrderItems { get; set; }

    protected override void OnModelCreating(ModelBuilder modelBuilder)
    {
        base.OnModelCreating(modelBuilder);

        modelBuilder.Entity<ReportOrder>(entity =>
        {
            // PK = OrderId (không dùng Id tự sinh — đây là Materialized View)
            entity.HasKey(e => e.OrderId);

            entity.Property(e => e.SubTotal).HasColumnType("decimal(18,2)");
            entity.Property(e => e.TotalCOGS).HasColumnType("decimal(18,2)");
            entity.Property(e => e.ShippingFee).HasColumnType("decimal(18,2)");
            entity.Property(e => e.TotalAmount).HasColumnType("decimal(18,2)");

            // Index chính cho báo cáo: lọc theo trạng thái + thời gian
            entity.HasIndex(e => new { e.Status, e.OrderedAt })
                  .HasDatabaseName("IX_ReportOrders_Status_OrderedAt");

            // Index cho báo cáo doanh thu: đơn hoàn thành trong kỳ
            entity.HasIndex(e => e.CompletedAt)
                  .HasDatabaseName("IX_ReportOrders_CompletedAt");

            // Index filter theo khách hàng
            entity.HasIndex(e => e.CustomerId)
                  .HasDatabaseName("IX_ReportOrders_CustomerId");
        });

        modelBuilder.Entity<ReportOrderItem>(entity =>
        {
            entity.HasKey(e => e.Id);

            entity.Property(e => e.UnitPrice).HasColumnType("decimal(18,2)");
            entity.Property(e => e.UnitCost).HasColumnType("decimal(18,2)");

            // Quan hệ với ReportOrder
            entity.HasOne(e => e.ReportOrder)
                  .WithMany(o => o.Items)
                  .HasForeignKey(e => e.OrderId)
                  .OnDelete(DeleteBehavior.Cascade); // Xóa Order → xóa luôn Items

            // Index chính: tổng hợp doanh thu / lợi nhuận theo sản phẩm
            entity.HasIndex(e => new { e.ProductId, e.OrderId })
                  .HasDatabaseName("IX_ReportOrderItems_ProductId_OrderId");
        });
    }
}

using CommonService.Events;

namespace ReportService.Consumers;

/// <summary>
/// Khi một đơn hàng được tạo → tạo bản sao local trong ReportDb để tổng hợp báo cáo sau.
/// </summary>
public class ReportOrderCreatedConsumer : IConsumer<OrderCreatedEvent>
{
    private readonly ReportDbContext _dbContext;

    public ReportOrderCreatedConsumer(ReportDbContext dbContext)
    {
        _dbContext = dbContext;
    }

    public async Task Consume(ConsumeContext<OrderCreatedEvent> context)
    {
        var ev = context.Message;

        // Idempotency: bỏ qua nếu đã có
        var exists = await _dbContext.ReportOrders.AnyAsync(o => o.OrderId == ev.OrderId);
        if (exists) return;

        var totalCOGS = ev.Items.Sum(i => i.UnitCost * i.Quantity);

        var reportOrder = new ReportOrder
        {
            OrderId       = ev.OrderId,
            CustomerId    = ev.CustomerId,
            CustomerName  = ev.CustomerName,
            SubTotal      = ev.SubTotal,
            TotalCOGS     = totalCOGS,
            ShippingFee   = ev.ShippingFee,
            TotalAmount   = ev.TotalAmount,
            Status        = "New",
            PaymentMethod = ev.PaymentMethod,
            OrderedAt     = ev.ProcessedAt,
            Items = ev.Items.Select(i => new ReportOrderItem
            {
                Id          = Guid.NewGuid(),
                OrderId     = ev.OrderId,
                ProductId   = i.ProductId,
                ProductName = i.ProductName,
                Quantity    = i.Quantity,
                UnitPrice   = i.UnitPrice,
                UnitCost    = i.UnitCost
            }).ToList()
        };

        await _dbContext.ReportOrders.AddAsync(reportOrder);
        await _dbContext.SaveChangesAsync();
    }
}

using CommonService.Events;
using MassTransit;
using ReportService.Data;

namespace ReportService.Consumers;

/// <summary>
/// Cập nhật trạng thái đơn hàng trong ReportDb khi OrderService thay đổi.
/// Quan trọng: Khi Status = "Completed" → ghi lại CompletedAt để tính doanh thu chính xác.
/// </summary>
public class ReportOrderStatusChangedConsumer : IConsumer<OrderStatusChangedEvent>
{
    private readonly ReportDbContext _dbContext;

    public ReportOrderStatusChangedConsumer(ReportDbContext dbContext)
    {
        _dbContext = dbContext;
    }

    public async Task Consume(ConsumeContext<OrderStatusChangedEvent> context)
    {
        var ev = context.Message;

        var reportOrder = await _dbContext.ReportOrders.FindAsync(ev.OrderId);
        if (reportOrder == null) return; // Bỏ qua nếu chưa có (race condition nhỏ, OK)

        reportOrder.Status = ev.NewStatus;

        // Chỉ ghi CompletedAt lần đầu khi chuyển sang Completed
        if (ev.NewStatus == "Completed" && reportOrder.CompletedAt == null)
            reportOrder.CompletedAt = ev.ProcessedAt;

        await _dbContext.SaveChangesAsync();
    }
}

```csharp
using Microsoft.EntityFrameworkCore;
using ReportService.Data;
using ReportService.Models.Response;
using ReportService.Services.Interface;

namespace ReportService.Services;

public class ReportServiceImpl : IReportService
{
    private readonly ReportDbContext _dbContext;

    public ReportServiceImpl(ReportDbContext dbContext)
    {
        _dbContext = dbContext;
    }

    public async Task<RevenueReportResponse> GetRevenueReportAsync(DateTime from, DateTime to)
    {
        from = DateTime.SpecifyKind(from, DateTimeKind.Utc);
        to   = DateTime.SpecifyKind(to,   DateTimeKind.Utc);

        // Chỉ tính đơn đã hoàn thành trong kỳ
        var completedOrders = await _dbContext.ReportOrders
            .Where(o => o.Status == "Completed"
                     && o.CompletedAt.HasValue
                     && o.CompletedAt >= from
                     && o.CompletedAt <= to)
            .ToListAsync();

        // Chi tiết theo ngày (cho biểu đồ)
        var daily = completedOrders
            .GroupBy(o => DateOnly.FromDateTime(o.CompletedAt!.Value.ToLocalTime()))
            .OrderBy(g => g.Key)
            .Select(g => new DailyRevenueItem
            {
                Date    = g.Key,
                Revenue = g.Sum(o => o.SubTotal),
                COGS    = g.Sum(o => o.TotalCOGS),
                Profit  = g.Sum(o => o.SubTotal - o.TotalCOGS),
                Orders  = g.Count()
            })
            .ToList();

        return new RevenueReportResponse
        {
            FromDate       = from,
            ToDate         = to,
            TotalRevenue   = completedOrders.Sum(o => o.SubTotal),
            TotalCOGS      = completedOrders.Sum(o => o.TotalCOGS),
            GrossProfit    = completedOrders.Sum(o => o.SubTotal - o.TotalCOGS),
            TotalOrders    = completedOrders.Count,
            DailyBreakdown = daily
        };
    }

    public async Task<List<TopProductResponse>> GetTopSellingProductsAsync(
        DateTime from, DateTime to, int top = 10)
    {
        from = DateTime.SpecifyKind(from, DateTimeKind.Utc);
        to   = DateTime.SpecifyKind(to,   DateTimeKind.Utc);

        // JOIN ReportOrderItems với ReportOrders đã Completed trong kỳ
        var result = await _dbContext.ReportOrderItems
            .Include(i => i.ReportOrder)
            .Where(i => i.ReportOrder!.Status == "Completed"
                     && i.ReportOrder.CompletedAt.HasValue
                     && i.ReportOrder.CompletedAt >= from
                     && i.ReportOrder.CompletedAt <= to)
            .GroupBy(i => new { i.ProductId, i.ProductName })
            .Select(g => new TopProductResponse
            {
                ProductId    = g.Key.ProductId,
                ProductName  = g.Key.ProductName,
                TotalQuantity = g.Sum(i => i.Quantity),
                TotalRevenue  = g.Sum(i => i.Quantity * i.UnitPrice),
                TotalProfit   = g.Sum(i => i.Quantity * (i.UnitPrice - i.UnitCost))
            })
            .OrderByDescending(p => p.TotalQuantity)
            .Take(top)
            .ToListAsync();

        // Gán Rank sau khi có kết quả
        for (int i = 0; i < result.Count; i++)
            result[i].Rank = i + 1;

        return result;
    }

    public async Task<List<TopProductResponse>> GetTopProfitableProductsAsync(
        DateTime from, DateTime to, int top = 10)
    {
        from = DateTime.SpecifyKind(from, DateTimeKind.Utc);
        to   = DateTime.SpecifyKind(to,   DateTimeKind.Utc);

        var result = await _dbContext.ReportOrderItems
            .Include(i => i.ReportOrder)
            .Where(i => i.ReportOrder!.Status == "Completed"
                     && i.ReportOrder.CompletedAt.HasValue
                     && i.ReportOrder.CompletedAt >= from
                     && i.ReportOrder.CompletedAt <= to)
            .GroupBy(i => new { i.ProductId, i.ProductName })
            .Select(g => new TopProductResponse
            {
                ProductId     = g.Key.ProductId,
                ProductName   = g.Key.ProductName,
                TotalQuantity = g.Sum(i => i.Quantity),
                TotalRevenue  = g.Sum(i => i.Quantity * i.UnitPrice),
                TotalProfit   = g.Sum(i => i.Quantity * (i.UnitPrice - i.UnitCost))
            })
            .OrderByDescending(p => p.TotalProfit)
            .Take(top)
            .ToListAsync();

        for (int i = 0; i < result.Count; i++)

    }

[Admin] PUT /payments/order/{orderId}/confirm
    └── PaymentService.ConfirmPaymentAsync()
            ├── Cập nhật Payment → Succeeded
            └── Publish PaymentSucceededEvent ──────────────────────────┐
                                                                         │
                └── (Chưa implement) OrderService consume PaymentSucceededEvent
                        └── Tự động chuyển Order → Processing

[Admin] PUT /orders/{id}/status  (chuyển sang Completed/Cancelled)
    └── OrderService.UpdateOrderStatusAsync()
            └── Publish OrderStatusChangedEvent ────────────────────────┐
                                                                         │
                ├── ProductService.OrderStatusChangedConsumer
                │       └── Trừ/Hoàn kho tương ứng
                │
                └── ReportService.ReportOrderStatusChangedConsumer
                        └── Cập nhật Status + CompletedAt trong ReportDb
```
